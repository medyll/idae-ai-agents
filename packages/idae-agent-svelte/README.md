# @medyll/idae-agent-svelte

This package provides an installable AI agent for Svelte projects, distributed via npm and runnable with `npx`. ok.

## Features
- Designed for Svelte project workflows
- Installable and executable via `npx @medyll/idae-agent-svelte`
- Integrates with GitHub for automation and documentation

## Getting Started

### Install dependencies
```sh
pnpm install
```

### Generate or update the agent
```sh
npx @medyll/idae-agent-svelte
```

### Run the agent
```sh
npx @medyll/idae-agent-svelte
```
